A Journey

to the Self

갤럭시 s9에 적용된 UI인 SE9을 소개하기위한 페이지 구축에 참혀하였다. 인덱스 페이지 각각의 키비쥬얼은 Shader와 css keyframe animation을 활용하였다. 이 페이지는 기존의 제품 스토리 위주의 디자인 삼성 컨텐츠와는 다르게 삼성 one UI의 기능에 대한 설명과 사용자의 인터페이스 사용 과정을 설명서처럼 사용자로 하여금 이해하기 쉽게 구현해야 하는 어려움이 있었다. 초반 시작부터 마이크로 모션이 들어갈 것을 세밀히 고려한 마크업이 이루어질 필요가 있었고 이를 통해 모션을 위한 마크업 설계에 대한 고민을 하게 되었다. 기존과는 다르게 가로 스크롤로 풀어낸 사이트였기 때문에 기존 세로 스크롤 타이밍에 맞춰 모션의 시작점을 캐치하던 작업을 모두 가로로 치환하여 작업할 필요가 있었다. UI의 동작과정을 나타낸 모션이 많아 세밀한 마이크로 모션이 많이 적용된 사이트이다.
